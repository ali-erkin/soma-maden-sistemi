# ⛏️ Soma Kömür Ocağı Yönetim Sistemi

Maden sahası için vardiya bazlı üretim takip ve raporlama sistemi.

## 🎯 Ne Yapıyor?

- 3 vardiya (Sabah/Öğle/Gece) takibi
- 5 araç için üretim ve yakıt hesabı
- Arıza tespiti ve raporlama
- Günlük ve haftalık otomatik rapor
- Web tabanlı, her cihazdan erişim

## 🛠️ Teknolojiler

- **Backend:** Python, Flask
- **Veritabanı:** SQLite
- **Frontend:** HTML, CSS
- **Raporlama:** Otomatik .txt dosyası

## 🚀 Kurulum

```bash
pip install flask
python app.py
```

Tarayıcıda `http://localhost:5000` aç.

## 📊 Özellikler

| Özellik | Açıklama |
|---|---|
| Vardiya Raporu | Araç bazlı üretim ve arıza takibi |
| Günlük Rapor | 3 vardiya özeti, hedef analizi |
| Haftalık Rapor | 7 günlük trend ve maliyet |

## 💡 Kullanım Senaryosu

Kağıt/Excel ile yapılan vardiya raporlamasını web tabanlı 
otomatik sisteme taşır. Saha şefi telefondan anlık takip edebilir.
